Modern Landing Template
Высокопроизводительный, адаптивный лендинг с плавными анимациями и поддержкой темной темы

Ообенности:

Fluid Animations: Использование Framer Motion для создания эффектов при скролле и интерактивных элементов
Dark/Light Mode
State Management: Глобальное состояние (тема, состояние меню, фильтры) управляется через Redux Toolkit
Performance Optimized: Использование стандартных оптимизаций Next.js
Fully Responsive

Технологический стек
Framework: Next.js
State: Redux Toolkit
Animation: Framer Motion
Styling: Tailwind CSS + Lucide Icons

Быстрый старт
git clone https://github.com/Sinonqw/example-landing.git
cd example-landing
npm install
npm run dev
